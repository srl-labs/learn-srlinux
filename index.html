    Learn Nokia SR Linux

    Nokia Service Router Linux (SR Linux) is a truly open network operating system (NOS) that makes your data center switching infrastructure more scalable, more flexible and simpler to operate.

    Get started

    What is Nokia SR Linux?

    SR Linux is a part of the Nokia Data Center Fabric solution that has been designed from the ground up to free your cloud builder networking teams to rapidly design and deploy, easily adapt and integrate and confidently operate and automate data center network fabrics: at scale.

    SR Linux is a truly open, extensible and resilient network operating system (NOS) is the foundation of this next generation solution, enabling scalability, flexibility and efficiency across your data center and cloud environments.

    Key features

    Standard linux kernel

    By using an unmodified Linux-based kernel as the foundation on which a suite of network applications is built, SR Linux provides reliability, portability and ease of application development.

    Modular architecture

    Designed from the ground up architecture that meets the demands of scalability, visibility and ease of operations.

    Proven protocol stack

    By using SR Linux, your data center operators benefit from the resilience, stability, scalability and proven interoperability of Nokia SR OS.

    Streaming telemetry

    SR Linux delivers an open and extensible infrastructure, which allows the retrieval of fine-grained system state, configuration setting, and a scalable interface to support more granular data with push-based streaming telemetry.

    Customizable CLI

    An advanced, Python-based CLI provides a flexible framework for accessing the systemâ€™s underlying data models. Operators can leverage CLI plugins to completely customize the way the CLI operates.

    Network Development Kit (NDK)

    The NDK enables operators to integrate their own and third-party applications into the system with all the same benefits as Nokia applications.

    Getting started with SR Linux

    SR Linux packs a lot of unique features that datacenter networking teams can leverage. Some of the features being truly new to the networking domain. The goal of this portal is to demonstrate these features and make SR Linux familiar to every visitor.

    We believe that learning by doing yields the best results. With that in mind we made SR Linux container image available to everybody without any registration or licensing.

    The tutorials on this portal will be augmented with lab scenarios that anyone can spin locally at any time.

    Open container image

    Run SR Linux anywhere where Docker is installed. Free and easy.

    Learn by doing

    Get to the grips of SR Linux features by doing real labs built with SR Linux container image.
